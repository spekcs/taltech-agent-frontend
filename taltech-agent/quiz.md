The user can request quizzes to be created based on the currently selected topic. 

On navigation the user can select whether they would like a test quiz or learning quiz. 

The quiz should use editor.js with custom tools. 

In first implementation all api queries will be mocked. 

For easy quizzes the backend will queries for quiz with selected topic and difficulty. 

If the quiz type is easy the answers will be textareas otherwise backend will give answer variants. 

On quiz complete the backend will evaluate the response. And give insights of topics that could be improved. 

The chattool can be called to explain the results of the quiz
