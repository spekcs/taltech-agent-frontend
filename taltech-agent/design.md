Stack: Vue 3 + Vite, Pinia for state, Vue Router, TailwindCSS
Core Design Language
Merge Notion's clean notebook aesthetic (serif-leaning typography, soft whites, subtle left-border accents, block-based content) with Napkin AI's spatial canvas interactions (freeform drag, zoom, node-based idea mapping, floating toolbars). The overall feel is "a student's intelligent notebook that thinks with you."
Use clean sans-serif fonts.
